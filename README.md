Find The Answer!

상식 퀴즈를 풀다 보면, 어느 순간부터 공포가 시작되는
반전형 공포 퀴즈 웹 프로젝트입니다.
처음에는 귀엽고 평범한 퀴즈처럼 보이지만,
선택과 진행에 따라 점점 불안하고 위협적인 연출이 등장합니다.

프로젝트 개요

이 프로젝트는 일반 상식 퀴즈와 공포 연출을 결합하여
사용자의 심리를 흔드는 인터랙티브 웹 경험을 제공하는 것을 목표로 합니다.

초반: 밝고 안전한 분위기의 상식 퀴즈

후반: 배경 변화, 사운드, 글리치, 점프스케어가 포함된 공포 퀴즈

엔딩: 선택에 따라 다른 결과와 강제 점프스케어 발생

단순히 문제를 푸는 것이 아니라
“끝까지 플레이할 것인가” 자체를 고민하게 만드는 구조입니다.

주요 기능
1. 사용자 닉네임 입력 및 플레이 기록

닉네임 중복 검사

플레이 시작 시점 저장

엔딩 도달 시 전체 플레이 시간 계산

2. 퀴즈 시스템

normal 퀴즈와 horror 퀴즈를 분리

문제 랜덤 섞기

일반 문제 → 공포 문제로 자연스럽게 전환

특정 문제는 선택 즉시 게임 오버 처리

3. 공포 연출 시스템

랜덤 점프스케어 이미지 및 사운드

화면 전체를 뒤덮는 강제 연출

글리치 효과, 화면 흔들림, 플래시 효과

오디오 레이어링을 활용한 저주파 사운드

4. 자동 트리거 공포 연출

특정 버튼 입력 시 강제 점프스케어

사용자의 의지와 무관한 연출로 불안감 극대화

기존 점프스케어 함수가 없을 경우 강제 대체 연출 실행

5. 엔딩 분기

게임을 끝내겠다는 선택

거부 시 즉시 공포 엔딩 발생 후 메인 화면으로 강제 이동

기획 의도

이 프로젝트는 단순한 공포 연출이 아니라
안전하다고 느끼는 상황이 무너질 때의 불쾌함과 공포를 목표로 합니다.

귀여운 디자인과 대비되는 갑작스러운 공포

선택해도 피할 수 없는 연출

사용자의 통제권을 빼앗는 구조

이를 통해
“웹에서도 충분히 심리적 공포를 만들 수 있다”는 것을 보여주고자 했습니다.

주의 사항

반드시 전체 화면 F11 모드로 플레이 권장

이어폰 또는 헤드셋 착용 시 공포 효과가 극대화됨

심약자는 플레이를 권장하지 않음
