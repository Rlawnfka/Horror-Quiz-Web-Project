Find The Answer!

Find The Answer!는 일반 상식 퀴즈 진행 중 점진적으로 공포 연출이 개입되는
반전형 인터랙티브 공포 퀴즈 웹 프로젝트입니다.
초반에는 밝고 단순한 퀴즈 형태로 시작하지만, 진행과 선택에 따라 분위기·연출·제어권이 변화하며
사용자에게 심리적 불안과 긴장을 유도합니다.

**프로젝트 개요
**
본 프로젝트는 퀴즈라는 안전한 구조에 공포 연출을 점진적으로 삽입하여
사용자의 심리를 흔드는 웹 경험을 제공하는 것을 목표로 합니다.

초반부: 밝은 분위기의 일반 상식 퀴즈

중·후반부: 배경 변화, 사운드, 글리치, 점프스케어가 포함된 공포 퀴즈

엔딩: 사용자의 선택에 따라 분기되며, 강제 공포 연출이 발생

단순한 문제 풀이가 아니라
“계속 진행할 것인가” 자체를 고민하게 만드는 구조를 핵심으로 설계했습니다.

기획 의도

본 프로젝트는 단순한 자극 위주의 공포가 아닌
안전하다고 인식한 구조가 붕괴되는 과정에서 발생하는 심리적 공포를 목표로 합니다.

귀여운 디자인과 대비되는 급격한 분위기 전환

선택해도 피할 수 없는 강제 연출

사용자의 통제권을 점진적으로 제거하는 구조

이를 통해
웹 환경에서도 충분히 강한 심리적 공포 연출이 가능함을 보여주고자 했습니다.
